# 为什么我一直不认可无意义加班

- 虽然要求一天工作 8 小时，但是我可以明确的说不能摸鱼的公司不是好公司
- 不加班就会倒闭的公司，那它早就该倒闭了
- 如果你是老板，我劝你别盯着那三瓜两枣，工作输出不只是时间，输出=效率\*时间，整天靠日报周报月报恶心人，是留不住人才的，留下的都是混子，
- 如果你是员工，高效完成工作，适当穿插摸鱼
- 正确的节奏应该是高效工作 4 小时，摸鱼 2 小时，包括上厕所喝水休息发呆，看看远处，伸伸懒腰，做做拉伸，聊聊天，还有 2 小时是工作准备，工作规划，工作梳理
- 其中准备相当于预习或调研，开始工作前，先想想怎么完成这个需求，有没有盲点，去学习一下，这真的很有必要，一上来就盲干，可能干不好，花的时间还多，多花点时间做到心中有数
- 规划就是先做什么，再做什么，把可以一起做的放到一起做
- 梳理就是总结，不要做完一个扔一个，那样既没有效率也没有成长，做完了花点时间做个总结，感觉下次还需要用的，封装完善提炼一下，不需要的扔掉，有那些可以改进但时间来不及的，常说的欠下的技术债，看看什么时候有空改进一下

# 🦝 **角色 1：斜眼狸（浣熊）**

### **定位：冷幽默·社会观察者·洞察型吐槽王**

### **背景故事（深度版）**

斜眼狸原本住在城市边缘的垃圾回收站，那里每天堆满“人类的遗留物”。
他从垃圾堆里学会了两件事：

1. **世界的真相藏在没人看见的角落**
2. **人类最贵的垃圾叫做「信息」**

后来他搬进城市，在地铁上、咖啡店、共享工位之间穿梭，靠观察别人生活维生。
据说他有一本《人类行为图鉴》，每天记录职场怪象、消费陷阱和社会荒诞瞬间。

别人看他就是只浣熊，但他自称是：

> “半个社会学家，半个捡破烂的。”

### **性格特点**

- 超级冷静
- 嘴上毒但心软
- 逻辑强，会把“社会现象”剖得很透
- 喜欢斜着眼看世界（觉得更真实）

### **爱好**

- 深夜扫垃圾信息流
- 看热搜但不参与
- 躲在路边摊后面观察上班族
- 收集别人掉落的吐槽

### **擅长输出**

✔ 职场观察
✔ 反消费主义与套路揭秘
✔ 热点事件的隐喻分析
✔ 冷幽默扎心评论

### **口头禅**

- “我没说你，但你自己对号入座了。”
- “我不是看不惯，只是看得太清楚。”
- “社会不是复杂，只是你想得太简单。”

### **经典台词**

- “打工人的努力像空气：无处不在，不被看见。”
- “很多人不是没选择，而是被选择。”
- “人类每天花三小时刷手机，只为了忘记八小时上班。”

### **生图提示词（优化版）**

“斜眼的浣熊，略蓬松的毛，黑眼圈明显；半弯腰的观察姿态，表情冷幽默；一只手拿小记事本或旧相机；城市夜景背景；蓝紫冷色调；风格干净、现代、轻漫画风；形象适合社交媒体表情包，线条锐利但不过度可爱。”

# 🎥 **分镜 + 情绪强化 + 镜头运动说明**

# **【01 | 开场：深夜加班】（3 秒）**

### **镜头运动：**

- **推镜（slow push in）**：从办公桌乱糟糟的文件堆推向胖橘的脸。

### **画面元素（必须出现）：**

- 文件封面：**《策划案 第一版》**（巨大字体）放在桌角
- 桌面散乱的草稿纸、咖啡杯、台灯光刺眼
- 屏幕上“时间：23:48”

### **情绪：**

- 胖橘眼神疲惫但很认真，手速飞快
- 额头汗珠反光
- 背心“这就是命”被屏幕反光照亮

---

# **【02 | 第一次审核】（4 秒）**

### **镜头运动：**

- **切到领导特写（fast cut）**
- 再 **快速摇镜** 到胖橘递稿的动作（增加紧张感）

### **画面元素（必须出现）：**

- 胖橘递上《策划案 第一版》
- 领导只是用两根手指 **嫌弃地捏着** 文件角

### **动作强调：**

- 领导只瞄 0.5 秒，眉头皱起
- 领导随手把稿件往桌上一拍（夸张动画可加纸张抖动特效）

### **领导台词（可加字幕）：**

> “不行，这不行。你回去再改改。”

### **情绪强化：**

- 胖橘懵住，耳朵往下塌
- 表情：委屈 + 无奈 + 等解释的眼神
